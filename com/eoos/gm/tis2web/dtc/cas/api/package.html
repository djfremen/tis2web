Provides the API for asynchronous DTC upload.

API Usage

The following code sample shows the usage of the API: 
	
	...
	byte[] dtc=...

	// 1. create instance of IDTCUpload
	IDTCUpload dtcUpload = DTCUploadFactory.createInstance(wrkDir, null);

	// 2. start upload
	Identifier id = dtcUpload.upload(Collections.singleton(dtc));

	//3. wait for termination
	while (!dtcUpload.finished(id)) {
                Thread.sleep(100);
    }
    ...
